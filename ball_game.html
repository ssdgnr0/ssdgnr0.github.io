Ball game

The ball game's game play is rather simple.
The objective: trying to keep the ball from falling to the very bottom of
the screen by balancing it on the orange bat,
while simultaneously trying to score points which is done by knocking
out the green balls.

Technologies

The project is made using C# and Microsoft's XNA game programming framework.

Design and structure

There are two versions of the ball game. The first is built purely on structs in addition to the primitive data types. The second is built on classes.
