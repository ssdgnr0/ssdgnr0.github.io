Snake

A simple snake clone made into a full featured game.
The project contains a title screen , menu screen ,a few confugurable options
via the settings screen, a 'save game' function and pause game screen as well a game over screen.
The game's level of difficulty changes as the player accumilates more points. This is a fully independent project and not a tutorial clone.

Technologies

The project is built on the python programming language and the panda3d game engine.

Design and structure

The design is very simple but still makes use of object oriented programming concepts.
I favour encapsulation over inheritance. for the most part.
